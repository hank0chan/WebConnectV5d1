1. 实体类
	User类：用户个人信息类
		id为自增唯一标识；username唯一；state为1说明用户在线；speak为0时说明被ROOT禁言；
	UserForm类：表单信息类
		大致如上。
	Message类：用户聊天记录类
		id为自增唯一标识；username非空，是发送者用户名；
	Root类：管理员个人信息类
		id为自增唯一标识；username非空；level为权限等级：1为最高级，能够执行禁言删除该用户等操作；
	Friends类：好友关系类
		id为自增唯一标识；usera和userb指向两个用户；isfriend为1说明两者是好友关系，否则为0；
